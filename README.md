# Aegis Frontend

Storm intelligence dashboard for Tampa Bay — Next.js + Tailwind CSS + React-Leaflet.

## Features

- Interactive Tampa Bay map with risk zones and incident markers
- Three-phase view: Pre-Storm → Active Storm → Post-Storm
- Real-time alert feed
- Field report submission
- Resource finder (shelters, supplies, charging stations)
- Family reunification portal

## Stack

- Next.js 14
- Tailwind CSS
- React-Leaflet
- TypeScript

## Setup

```bash
npm install
cp .env.example .env.local  # Add backend URL
npm run dev
```
